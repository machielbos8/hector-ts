# Examples

Eight worked examples illustrating the Hector v3.0 workflow.

| Example | Topic |
|:--- |:--- |
| [ex1](ex1/README.md) | Synthetic GNSS: removeoutliers → estimatetrend → PSD |
| [ex2](ex2/README.md) | Monthly sea-level data from Cascais tide gauge |
| [ex3](ex3/README.md) | Synthetic coloured noise with simulatenoise |
| [ex4](ex4/README.md) | Post-seismic relaxation |
| [ex5](ex5/README.md) | Offset detection on a real GNSS station |
| [ex6](ex6/README.md) | Multi-station offset detection from NGL tenv files |
| [ex7](ex7/README.md) | Piecewise linear (multi-trend) estimation |
| ex8 | Toeplitz factorisation: Levinson vs. Generalised Schur (Jupyter notebook) |

Run each example from its own directory with Hector installed and active.
